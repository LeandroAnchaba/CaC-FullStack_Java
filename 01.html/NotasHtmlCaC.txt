Etiquetas semanticas: no modifican la pagina, sirven para dividir los contenidos dentro del body
Creadeas desde la etiqueta div (divisor)
Dentro del body  se divide en: Header, main, footer
etiqueta "nav" para hacer un menu
etiqueta "article" para hacer un articulo
Las propiedades de las etiquetas van siempre en la apertura ej: <div --propiedad--> contenido </div>
Para luego poder usar javaScrip se usa el identificador "id"
    y para java hay que usar el atributo "name" en las etiquetas.
    conviene llamar al "id" y al "name" de la misma manera
